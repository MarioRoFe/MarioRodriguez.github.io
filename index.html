Predicción de precios inmobiliarios

Web Scraping, Limpieza de datos, EDA, Visualización y Análisis de datos, Machine Learning, Desarrollo de aplicación Web.

Tecnologías usadas
Python: Scrapy, Selenium, Scikit-Learn, Pandas, Seaborn, Streamlit.

Este proyecto fue realizado para uso de una agencia de bienes raíces ubicada en la ciudad de Oaxaca de Juárez.
Mediante el uso de un modelo de Machine Learning, se puede predecir el precio de una casa ubicada en la ciudad o tomar un precio conocido y compararlo con el precio de mercado predicho por el modelo para saber si este es competitivo. El propósito de este proyecto es ayudar a realizar un análisis de precios más preciso y con ello ajustar sus estrategias de ventas.

Análisis de ventas

Limpieza de datos, EDA, Visualización y Análisis de datos
SQL, Python: pandas, sqlalchemy, matplotlib, seaborn, squarify

Se analizan los datos de venta de una empresa con el objetivo de identificar nuevas áreas de oportunidad que deban ser aprovechados o posibles problemas y riesgos que deban ser solucionados. Se encontró que algunos artículos generan pérdidas debido a sus altos costos, por lo que es necesario examinar la estructura de costos e incluso plantear el dejar de vender artículos de algunas categorías.
