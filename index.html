Análisis Predictivo del Precio de Diamantes mediante Modelos de Machine Learning

EDA, Visualización de datos, Machine learning
Python: numpy, pandas, scikitlearn, statsmodels, matplotlib, seaborn

Se entrena un modelo de machine learning para predecir el precio de diamantes con base en sus principales características como corte, color, longitud, etc. Se evalua el desempeño de modelos como regresión lineal multiplie, regresión lasso, regresión ridge y regresión polinomal. El modelo que mejor se ajusta a estos datos en particular es la Regresión Polinomial con Regularización Lasso con un valor óptimo de alpha = 0.00336
